# Vibe Coder — Portfolio

Сайт-портфолио вайб-кодера: лендинги, интерфейсы, digital-проекты.  
Тёмная тема, анимации, современный визуал.

## Превью

Первый экран с эффектом печатной машинки, bento-сетка навыков, карусель отзывов и плавные reveal-анимации при скролле.

## Структура проекта

```
my_Portfolio/
├── index.html                 # Главная страница
├── styles.css                 # Основные стили
├── css/
│   └── magic-bento.css        # Стили bento-сетки навыков
├── js/
│   ├── hero-typewriter.js     # Эффект печатной машинки (hero)
│   ├── magic-bento.js         # Логика bento-карточек
│   └── stagger-testimonials.js# Карусель отзывов
├── images/                    # Изображения (hero и др.)
├── TZ.md                      # Техническое задание
└── README.md
```

## Технологии

| Категория | Что используется |
|-----------|-----------------|
| Разметка | HTML5, семантические теги |
| Стили | CSS3, custom properties, clamp(), grid, flexbox |
| Анимации | GSAP 3.12, IntersectionObserver, CSS transitions |
| Шрифты | Google Fonts — Syne (display), Literata (body) |
| Деплой | Статический сайт, без сборщиков |

## Разделы сайта

- **Hero** — первый экран с typewriter-заголовком и CTA
- **Обо мне** — позиционирование и список направлений
- **Что я делаю** — bento-сетка из 6 карточек с glow-эффектом
- **Как я работаю** — 3 шага подхода
- **Избранные проекты** — карточки кейсов
- **Отзывы** — горизонтальная карусель
- **Почему со мной** — ценностное предложение
- **Инструменты** — бейджи стека
- **Для кого** — целевая аудитория
- **Манифест** — цитата
- **CTA + Контакты** — Telegram, email, соцсети

## Запуск

Проект полностью статический — достаточно открыть `index.html` в браузере или запустить любой локальный сервер:

```bash
# Python
python -m http.server 8000

# Node.js (npx)
npx serve .

# VS Code — расширение Live Server
```
